ABOUT US

This website is a student collaboration project for the internet programming course at ASU "CSE341", this website is meant to emulate a plane reservation website where the user is able to reserve a seat on any plane thats available to got to any destination , the user is then able to get an ETA on when that plane is set to arrive and the route its going to take , the user is also able to give a review to the carrier of said plane after the trip is concluded.
